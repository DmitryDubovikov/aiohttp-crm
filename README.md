# aiohttp-CRM

Activate virtual env:

    pipenv shell
    pipenv install

## API endpoints 

| Endpoint    | Method | Note                |
|-------------|--------|---------------------|
| /add_user   | POST   | provide email       |
| /list_users | GET    | provide credentials |
| /get_user   | GET    | provide user id     |
| /docs       | GET    | Swagger UI          |
| /docs/json  | GET    | Swagger json        |
